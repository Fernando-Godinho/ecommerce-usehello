Como publicar suas alterações no ar? 🚀

Este painel armazena temporariamente suas modificações no seu navegador. Para que os clientes vejam as mudanças:

  1. Realize todas as edições desejadas no catálogo, cupons e timers.
  2. Acesse a aba "Publicar Alterações" no menu lateral.
  3. Baixe os arquivos products.js e config.js.
  4. Substitua os arquivos correspondentes na pasta do projeto no Git e faça o commit/push.
  5. O Easypanel identificará a alteração e subirá a nova versão da loja em segundos!
